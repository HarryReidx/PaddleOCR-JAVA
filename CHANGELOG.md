# Changelog

## 1.0.0 - 2026-03-31

### Added

- 新增同步识别、异步入队、任务详情、任务停止、队列状态与 SSE 实时通知能力
- 新增 OCR 结果多区域展示：任务信息、OCR 原始文本、字段提取、JSON 输出
- 新增多引擎对比能力：`paddleocr-cpu`、`qwen-vl`、`glm-ocr`、`paddleocr-vl`
- 新增前端图片扰动测试能力：模糊、倾斜、失真、遮挡、水渍、反光、褶皱
- 新增遮挡拖拽定位能力
- 新增 Docker 镜像构建与 Harbor 推送支持

### Changed

- 控制台标题统一为“清云OCR控制台”
- 上传区与识别结果区布局重构，整体高度和滚动逻辑统一
- 识别结果区调整为四卡片结构，并增加复制能力
- 图片处理悬浮面板重构为毛玻璃样式，支持收起展开
- 引擎选择器从原生下拉调整为自定义下拉面板，统一视觉样式
- 模糊默认强度调整为 8%
- 遮挡 100% 强度调整为纯黑不透明
- 前端构建产物同步策略明确为复制 `frontend/dist/*` 到后端 `static` 根目录

### Fixed

- 修复任务状态与结果展示不一致问题
- 修复任务队列、实时通知、识别结果区域高度与滚动限制问题
- 修复同步识别时缺少 Loading 态的问题
- 修复前端静态资源错误嵌套到 `static/dist` 的问题
- 修复悬浮面板遮挡图片预览与滑块显示不全的问题
